# Art Portfolio Board

Production-ready animated portfolio interface inspired by handcrafted sketch composition.

## Stack

- Vite (build + dev server)
- GSAP + ScrollTrigger (motion and scroll choreography)
- AOS (progressive reveal)
- Lottie-web (hero animation)
- Particles.js (subtle background depth)
- Feather Icons (consistent icon system)

## Quick Start

1. Install dependencies:
   npm install
2. Run development server:
   npm run dev
3. Create production build:
   npm run build
4. Preview production build locally:
   npm run preview

## Production Notes

- Reduced-motion users are automatically respected.
- Images are lazy-loaded and async-decoded.
- Decorative animations are subtle to preserve UX quality.
- The project uses a single coherent visual system for colors, borders, motion, and iconography.

## File Structure

- index.html: Document shell and layout
- styles.css: Design system and component styling
- script.js: Motion orchestration and runtime behavior
- vite.config.js: Build and preview config

## Deployment

- Deploy the contents of `dist/` generated by `npm run build`.
- `base` is set to `./` in Vite config for static hosting compatibility.
